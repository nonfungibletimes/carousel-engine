# Assets

Drop brand assets here. These are referenced by templates using relative paths `../../assets/`.

## PAI3 Assets
- `bg-template.png` — dark background texture
- `pai3-logo.png` — PAI3 logo (square)
- `power-node.png` — Power Node product photo

## Adding New Brand Assets
1. Drop files into this folder
2. Reference them in your template as `../../assets/filename.png`
3. Commit and push — they'll be available to all templates

> **Note:** Large binary assets (product photos, backgrounds) are committed directly to the repo.
> Use reasonably sized files — compress PNGs before adding if >2MB.
